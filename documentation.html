Data Sources

This dashboard compiles data from multiple reliable sources to analyse mobile-phone enforcement trends across Australia. The selected datasets support long-term trend analysis, enforcement method comparison, and state-level evaluation.

Mobile Phone Enforcement Statistics — Australian Road Safety Agencies

Core dataset containing annual enforcement counts for police-issued fines and camera-detected offences.

AI-Based Camera Detection Records — State Transport Departments

Dataset includes detection counts from AI-enabled mobile phone cameras introduced from 2020 onward.

Driver Population & Licensing Data — Australian Bureau of Statistics

Provides the number of registered drivers per state, used to normalise enforcement values for fair comparison.

Processed Datasets

All datasets were cleaned, normalised, and formatted to ensure consistency across states and enforcement methods. These processed files were used directly in the dashboard.
